Урок з Python - Розділ 6: Аналіз даних за допомогою словників

Створення словників Доступ до даних за ключем Оновлення та видалення записів Вкладені словники Аналіз даних зі словниками Довідкові документи

1. Створення словників

Словники в Python зберігають дані у вигляді пар "ключ-значення". Ключі повинні бути унікальними, і через них можна швидко отримувати значення.

студент = {"ім'я": "Олена", "вік": 20, "курс": "Python"}
оцінки = {"математика": 90, "англійська": 85, "фізика": 95}

2. Доступ до даних за ключем

Для отримання значення зі словника використовують ключ. Якщо ключ відсутній, можна використовувати метод get(), щоб уникнути помилки.

ім'я = студент["ім'я"] # Отримуємо значення за ключем
курс = студент.get("курс", "Невідомий") # Безпечний доступ

3. Оновлення та видалення записів

Щоб змінити значення у словнику, достатньо вказати нове значення за існуючим ключем. Для видалення використовуються методи pop() або del.

студент["вік"] = 21 # Оновлюємо значення
видалений_курс = студент.pop("курс") # Видаляємо ключ "курс"
del студент["вік"] # Видаляємо ключ "вік"

4. Вкладені словники

Словники можуть містити інші словники як значення. Це корисно для складних структур даних, наприклад, зберігання інформації про студентів і їх оцінки.

студенти = {
    "Олена": {"вік": 20, "курс": "Python"},
    "Андрій": {"вік": 22, "курс": "Математика"}
}

курс_олени = студенти["Олена"]["курс"]

5. Аналіз даних зі словниками

Python надає можливості для аналізу даних у словниках. Наприклад, можна підрахувати середній бал студента, використовуючи значення словника.

оцінки = {"математика": 90, "англійська": 85, "фізика": 95}
середній_бал = sum(оцінки.values()) / len(оцінки)
print("Середній бал:", середній_бал)
