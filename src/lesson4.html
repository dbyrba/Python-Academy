Урок з Python - Розділ 4: Ігри з функціями

1. Створення функцій

Функції дозволяють структурувати код і використовувати його повторно. Щоб створити функцію в Python, використовуйте ключове слово def, ім'я функції та круглі дужки. Код, що знаходиться всередині функції, виконується при виклику цієї функції.

def привітання():
    print("Привіт! Ласкаво просимо до нашої гри!")

2. Параметри та аргументи

Функції можуть приймати параметри, які дозволяють передавати дані у функцію під час її виклику. Це робить функцію гнучкою і здатною працювати з різними значеннями.

def привітання(ім'я):
    print("Привіт,", ім'я, "! Ласкаво просимо до гри!")

привітання("Олексій")

3. Повернення значень

Функція може повернути результат, використовуючи ключове слово return. Це дозволяє передавати обчислене значення з функції в програму.

def додати(а, б):
    return а + б

результат = додати(3, 5)
print("Результат:", результат)

4. Використання випадкових значень

Для створення випадкових чисел в Python можна використовувати модуль random. Це корисно для створення ігор, де потрібні випадкові значення.

import random

випадкове_число = random.randint(1, 10)
print("Випадкове число від 1 до 10:", випадкове_число)

5. Гра “Вгадай число”

Давайте створимо просту гру "Вгадай число", в якій користувач повинен вгадати випадкове число від 1 до 10. Гра буде використовувати функції, випадкові значення і цикли для організації ігрового процесу.

import random

def гра_вгадай_число():
    число = random.randint(1, 10)
    спроба = 0

    while True:
        введене_число = int(input("Вгадай число від 1 до 10: "))
        спроба += 1

        if введене_число == число:
            print("Вітаю! Ви вгадали число за", спроба, "спроб.")
            break
        elif введене_число < число:
            print("Загадане число більше.")
        else:
            print("Загадане число менше.")

гра_вгадай_число()
