Урок з Python - Розділ 2: Розробка алгоритмів з умовними операторами

Логічні умови Складені булеві вирази Ланцюгові умови Вкладені умови Вироблення рекомендацій Довідкові документи

1. Вступ до логічних умов

Умовні оператори у Python допомагають приймати рішення залежно від результату певного виразу. Основний оператор для створення умов — if. Логічні умови дозволяють перевірити, чи є вираз істинним чи хибним, і, залежно від цього, виконати певні дії.

2. Складені булеві вирази

У Python можна об’єднувати кілька логічних умов, використовуючи оператори and, or та not. Це дозволяє створювати більш складні умови.

age = 20
is_student = True

if age >= 18 and is_student:
    print("Доступ дозволено.")
else:
    print("Доступ заборонено.")

3. Ланцюгові умови

Ланцюгові умови дозволяють обробляти кілька можливих сценаріїв за допомогою операторів if, elif і else. Це особливо корисно, коли є декілька варіантів розвитку подій.

grade = 85

if grade >= 90:
    print("Оцінка: A")
elif grade >= 80:
    print("Оцінка: B")
elif grade >= 70:
    print("Оцінка: C")
else:
    print("Оцінка: D")

4. Вкладені умови

Вкладені умови дозволяють перевіряти додаткові умови всередині інших умов. Це корисно, коли для прийняття рішення потрібно врахувати кілька різних факторів одночасно.

is_registered = True
has_ticket = True

if is_registered:
    if has_ticket:
        print("Вхід дозволено.")
    else:
        print("Квиток потрібен для входу.")
else:
    print("Будь ласка, зареєструйтесь.")

5. Вироблення рекомендацій

На основі умов можна створювати рекомендації або приймати рішення в коді, враховуючи кілька факторів. Наприклад, можна поєднувати умови для створення рекомендацій для користувача, ґрунтуючись на його відповіді або діях.

age = 16
has_permission = False

if age >= 18:
    print("Ви можете взяти участь.")
elif age >= 16 and has_permission:
    print("Ви можете взяти участь з дозволом.")
else:
    print("Ви не можете взяти участь.")
