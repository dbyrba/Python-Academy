Урок з Python - Розділ 5: Автоматизація завдань зі списками

Створення списків Додавання та видалення елементів Перегляд списків Сортування списків Аналіз даних у списках Довідкові документи

1. Створення списків

Списки в Python - це тип даних, який дозволяє зберігати колекцію елементів. Їх можна створити за допомогою квадратних дужок, і вони підтримують різні типи даних.

міста = ["Київ", "Львів", "Одеса", "Харків"]
числа = [1, 2, 3, 4, 5]

2. Додавання та видалення елементів

Ми можемо додавати елементи до списку за допомогою методу append(), а видаляти - за допомогою методів remove() або pop().

міста.append("Дніпро") # Додаємо елемент
міста.remove("Львів") # Видаляємо елемент за значенням
видалений_елемент = міста.pop() # Видаляємо останній елемент

3. Перегляд списків

Списки можна переглядати, використовуючи цикли, що дозволяє обробляти кожен елемент у списку. Це корисно для аналізу та обробки даних.

for місто in міста:
    print(місто)

4. Сортування списків

Python дозволяє сортувати списки за допомогою методу sort() для змінних списків або функції sorted() для створення відсортованих копій списків.

числа = [5, 3, 8, 1, 9]
числа.sort() # Сортування списку
print("Відсортовані числа:", числа)

5. Аналіз даних у списках

Можна використовувати різні методи для аналізу даних у списках, як-от min(), max(), sum(), щоб отримувати важливу інформацію.

числа = [1, 2, 3, 4, 5]
print("Максимальне значення:", max(числа))
print("Мінімальне значення:", min(числа))
print("Сума:", sum(числа))
