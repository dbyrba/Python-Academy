Урок з Python - Розділ 3: Моделювання явищ за допомогою циклів

Імпорт модулів Цикл while Повторювання Потік керування циклом Моделювання популяцій Довідкові документи

1. Вступ до циклів у Python

Цикли дозволяють повторювати певні дії декілька разів, що є особливо корисним для моделювання різних явищ. У цьому розділі ми розглянемо базові цикли в Python, такі як while, та методи керування потоком всередині цих циклів.

2. Імпорт модулів

Для моделювання більш складних явищ можна імпортувати спеціальні модулі Python, наприклад, math або random. Це розширює можливості коду і додає додаткові функції, такі як генерація випадкових чисел чи обчислення математичних виразів.

import random
import math

3. Використання циклу while

Цикл while повторює блок коду, поки умова істинна. Це основний цикл, який часто використовується для моделювання процесів, що потребують невизначену кількість повторень.

count = 0

while count < 5:
    print("Повторення №", count)
    count += 1

4. Потік керування циклом

Python дозволяє керувати потоком циклу за допомогою команд break та continue. break виходить з циклу, а continue пропускає ітерацію та переходить до наступної.

count = 0

while count < 10:
    if count == 5:
        break
    if count % 2 == 0:
        count += 1
        continue
    print("Непарне число:", count)
    count += 1

5. Моделювання популяцій

Цикли можна використовувати для моделювання росту популяцій або подібних процесів. Наприклад, використовуючи цикл while, ми можемо моделювати збільшення популяції, враховуючи певний коефіцієнт росту та випадкові коливання.

import random

population = 1000
growth_rate = 1.02
years = 0

while population < 2000:
    population *= growth_rate
    population += random.randint(-10, 10) # випадкове коливання
    years += 1
print("Популяція досягла 2000 за", years, "років.")
