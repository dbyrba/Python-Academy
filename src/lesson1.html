Урок з Python - Розділ 1: Обчислювальне мислення зі змінними

Платформа програмування Виконання програми Змінні Арифметичні вирази Шаблони веб-сторінок Довідкові документи

1. Вступ до обчислювального мислення

Обчислювальне мислення — це спосіб розв’язання проблем за допомогою декомпозиції, розпізнавання шаблонів, абстракції та алгоритмічного мислення. У цьому уроці ми розглянемо, як використовувати змінні, платформи програмування та арифметичні вирази для вирішення завдань у Python.

2. Що таке змінна?

Змінна — це контейнер, що зберігає значення, яке можна змінити у процесі виконання програми. У Python змінні можуть бути різних типів, наприклад:

  • ● Числові значення: цілі числа (int), дробові числа (float).
  • ● Рядки: текстова інформація.
  • ● Логічні значення: True або False.
name = "Anna"
age = 25
is_student = True

3. Виконання програми на платформі програмування

Для виконання коду Python можна використовувати інтегроване середовище розробки (IDE), як-от IDLE або Jupyter Notebook. Записавши код, ви можете натиснути "Виконати" для перегляду результату.

print("Hello, " + name + "!")
# Результат: Hello, Anna!

4. Арифметичні вирази у Python

Python підтримує стандартні арифметичні операції: додавання, віднімання, множення та ділення. Ви можете комбінувати змінні у виразах для обчислення результату.

price = 100
discount = 20
final_price = price - discount
print("Знижена ціна:", final_price)
# Результат: Знижена ціна: 80

5. Шаблони веб-сторінок з використанням змінних

Шаблони веб-сторінок можуть містити змінні для динамічної зміни контенту. Наприклад, використовуючи HTML, можна вставляти змінні у веб-сторінку.

<h1>Вітаємо, {{ name }}!</h1>
<p>Ваш вік: {{ age }} років.</p>
